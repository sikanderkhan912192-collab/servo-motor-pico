# Servo Motor Control using Raspberry Pi Pico

## 📌 Project Overview
This project demonstrates how to control a servo motor using Raspberry Pi Pico and MicroPython.

## 🧰 Components Used
- Raspberry Pi Pico
- Servo Motor (SG90)
- Jumper Wires

## ⚙️ Working
The servo rotates between 0°, 90°, and 180° using PWM signal.

## 🔌 Connections
- Red → VBUS (5V)
- Brown → GND
- Orange → GP15
